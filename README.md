# ACM (FacingApp) - Project Overview

ACM is an Expo-based React Native application integrated with HyperSnap SDK for identity verification and Firebase for messaging. This project is configured to support advanced native features and custom build processes.

## 📁 Project Structure

Detailed overview of the project's directory structure and core files:

### Root Directories

- **`/app`**: Contains the main application routing and screen logic using Expo Router.
  - `_layout.tsx`: Root layout configuration.
  - `index.tsx`: Main entry point/screen.
  - `/locales`: Localization files (e.g., `vi.ts`).
- **`/assets`**: Static files like images, fonts, and icons.
- **`/components`**: Reusable UI components.
  - `/ui`: Basic UI building blocks.
  - `security-check-wrapper.tsx`: Component for handling security and jailbreak detection.
- **`/hooks`**: Custom React hooks (e.g., `use-push-notifications.ts`, `use-theme-color.ts`).
- **`/constants`**: Shared constants and styling themes (e.g., `theme.ts`).
- **`/plugins`**: Custom Expo config plugins for native Android and iOS modifications.
  - `withReleaseSigning.js`: Handles APK signing configurations.
  - `withRootGradleConfig.js`: Customizes the root `build.gradle` for the Android project.
- **`/scripts`**: Utility scripts for project maintenance (e.g., `reset-project.js`).
- **`/android` & `/ios`**: Native project directories (generated by Expo Prebuild).

### Core Configuration Files

- **`app.json`**: Main Expo configuration file, including plugin registrations and native details.
- **`package.json`**: Defines dependencies and project scripts.
- **`RELEASE_GUIDE.md`**: Detailed instructions for the Android APK release process.
- **`release-android.sh`**: Automated script for building and releasing Android APKs.
- **`google-services.json` & `GoogleService-Info.plist`**: Firebase configuration files for Android and iOS.

## 🚀 Getting Started

### Prerequisites

- Node.js & npm/yarn
- Expo CLI
- Android Studio / Xcode for native builds

### Installation

1. Clone the repository.
2. Install dependencies:
   ```bash
   npm install
   ```

### Running the App

- **Development Mode**:
  ```bash
  npx expo start
  ```
- **Android**:
  ```bash
  npm run android
  ```
- **iOS**:
  ```bash
  npm run ios
  ```

## 🛠 Features & Integration

- **HyperSnap SDK**: Integrated for secure document and face capture.
- **Firebase Messaging**: Supported via `@react-native-firebase/messaging` for push notifications.
- **Security Checks**: Jailbreak and root detection using `jail-monkey` and custom wrappers.
- **Custom Native Config**: Leverages Expo's plugin system for deep native customization without manual edits to `android/` or `ios/` folders.
